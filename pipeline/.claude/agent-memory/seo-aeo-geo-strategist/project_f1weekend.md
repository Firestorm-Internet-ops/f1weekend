**F1Weekend** is a content + booking platform for F1 race travel. The SEO/AEO/GEO Strategist agent generates editorial race guide JSON consumed by the Next.js app.

**Standard JSON output schema:**
- hero_title, hero_subtitle
- guide_intro (2-3 sentences)
- why_city_text (150-200 words)
- highlights_list (5-6 bullets)
- circuit_facts (Lap Record, Circuit Length, Turns, DRS Zones)
- faq_items (6 questions: transport, arrival timing, between sessions, food, weather, booking tips)

**Why:** Content must perform on Google (SEO), answer engines (AEO), and generative AI experiences (GEO). Numbers in first sentences, active voice, no hedging.

**How to apply:** Always lead FAQ answers with the most important fact or number. Use local insider voice. Keep F1 session gaps in mind for activity recommendations. Night race circuits (Qatar, Abu Dhabi, Singapore) warrant specific atmosphere/lighting mentions.
